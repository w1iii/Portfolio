# Lui Franz A. Lomugdang — Portfolio

## Setup

```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (icons)

## Structure
- `app/page.tsx` — Main portfolio page (all sections)
- `app/globals.css` — Global styles, animations, hover effects
- `app/layout.tsx` — Root layout with metadata
- `tailwind.config.js` — Custom colors and fonts

## Customization
- Swap social media icon URLs in `page.tsx` → `SocialIcon` components
- Add your real profile photo: replace the initials avatar with an `<Image>` tag
- Update project data in the `projects` array at the top of `page.tsx`
- Update skills in the `skills` array
